 Hak - Hak User
1
SuperUser
Kelola Admin - Tambah/Edit Kategori - Tambah/Edit Dinas - Lihat Info Akun Sendiri - Lihat Info User - Lihat Daftar User Yang Mengikuti Post.
2
Admin Kominfo (Umum)
Hak SuperUser, tetapi tidak bisa mengelola admin kominfo lain - Disposisi Post - Edit/Delete Post, Tambah Jawaban dan Edit/Delete Jawaban untuk Post dengan Dinas Kominfo.
3
Admin Dinas Lain
Lihat Info Akun Sendiri - Lihat Info User - Lihat Daftar User Yang Mengikuti Post - Edit/Delete Post, Tambah Jawaban dan Edit/Delete Jawaban untuk Post dengan Dinas yang Sesuai dengan Akun.
4
Regular User (user)
Lihat Info Akun Sendiri - Tambah Post - Edit/Delete Post - Ikut (Simpan) Post.
5
Guest (pengunjung yang belum login)
Lihat Post dan Jawaban - Cari Post
 Aturan Buat Post
1
Cari post yang ingin dibuat, mungkin post sudah ada.
2
Gunakan judul dengan struktur kalimat tanya (5W/1H + S + P + O).
Contoh judul: Apakah ada subsidi untuk pembelian panel surya?
 Disposisi Post
Disposisi post dilakukan oleh admin kominfo jika post dinilai terlalu spesifik dan perlu dijawab oleh dinas terkait.
Mengganti Dinas Post
Post yang baru diajukan otomatis masuk ke dinas kominfo(umum). Post hanya bisa dipindahkan (didisposisi) oleh admin kominfo.
